Secret List Tutorial

Learn how to get the Secret List application and use it!

PREREQUISITES
Please note that this tutorial is based on Windows and will not work on mobile devices or Apple/Mac devices. Before starting, you must have the following. If you have any questions, you will gain support after you are approved and you have downloded the application.
• Basic computer knowledge- e.g. how to extract ZIP files.
• Know how to read English and follow instructions.
• A working email that you are able to check frequently.
• Java 8 or above, installed. (If you don't have it, download here.)
• You agree not to do anything stupid, like leaking the application and its information, changing the code in the program, etc.
• Common sense.


HOW TO DOWNLOAD AND INSTALL
1) Go to the Download page of this website.
2) Request the application download.
3) Once your request gets accepted, click on the link that was sent to the email you provided.
4) Download the application from the link provided.
5) Extract the ZIP file into a folder in your documents, on your desktop, etc.
6) There should be 2 files: SecretList.jar and run.bat. Click on the run.bat and a command prompt will pop up.
7) The application will load. If you see an error with something about "EXPIRED PLEASE UPDATE", just re-download the application and re-do the steps. It means you need to update the application (more about common errors at the bottom).
8) If loading is successful, you will be asked for your username, which you can find in the email you received. If your username is valid, a password will be asked for, which is also included in the email.
9) If authentication is successful, you have successfully downloaded, installed, and set-up the applcaition.
10) Read the README FILE, and keep on reading this page for information on application usage.


USAGE

COMMANDS
The Secret List Application has several commands: "List", and "More".
List: Lists everything from the Secret List.
More: Displays more information about the things from the secret list. Execute the command and wait until you are asked to enter an item ID. Extra information about the "thing" with the specified ID will be shown.

AUTHENTICATE
For security, the application will ask you to authenticate every time you run a command. It will ask for your username, then your password. Your credentials should be in the email you receive. Simply type your username/password and press enter.

VIEWING THE LIST
Well, it should be pretty self-explanatory once you extract the ZIP file. There is a BAT file called "List All Phrases". Simply click on that file and follow the steps in the "Authenticate" section. Then, the phrases should start popping up.

COMMON ERRORS

"INVALID TOKEN"
You will get this error if your "token" is incorrect. This means that you may have tampered with the program, or modified a BAT file.

"INCORRECT TIME"
Your device date is set incorrectly, or it's inaccurate. Don't try to manipulate your system time.

"EXPIRED"
Your application has expired, and you must reply to your approval email and request the updated version. You will not be contacted when applications expired, and you won't receive the update unless you ask.

"NO COMMAND SPECIFIED"
You did not specify a command. There is something wrong with your BAT file, it's probably because you tampered/edited it.

"INCORRECT CREDENTIALS, TRY AGAIN"
The most common error- your username and/or password is incorrect. Your credentials are specified in your approval email, as stated in the "Authenticate" section of this tutorial.

"POSSIBLE ACCOUNT COMPROMISE (BANNED)"
Your account may have been compromised (hacked, leaked, stolen) and you have been banned for security reasons. Reply to your approval email if you believe that this is incorrect.

"404"
The application could not find a text file online. Did you tamper with the JAR?

"CONNECTION FAILED (INTERNET CONNECTION INVALID)"
Your Internet is not working properly or you are not connected at all.

"INTERRUPTEDEXCEPTION"
This error is pretty rare. Restarting the application should fix this.

"INVALID COMMAND"
Did you edit the BAT file?
